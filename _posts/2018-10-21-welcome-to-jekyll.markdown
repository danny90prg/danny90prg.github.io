작성된 포스트는 
`YEAR-MONTH-DAY-title.MARKUP` 포맷의 파일명을 갖으며, `_posts` 디렉토리에 담깁니다.
빌드하고 싶으면 `jekyll serve` 명령어로 빌드합니다.
Jekyll은 스니펫을 제공합니다. 
사용예
{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
